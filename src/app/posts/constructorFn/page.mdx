> 💡 자바스크립트 정리 생성자 함수 정리

### 객체 리터럴

- 객체 리터럴 `{...}` 을 사용하면 객체를 쉽게 만듦

```jsx
let user = {
  name: "Mike",
  age: 30,
};
//객체 리터럴
```

### 생성자 함수

- 비슷한 객체를 여러 개 만들어야 할 상황에 사용
- 붕어빵 틀 재료를 넣어주고 찍어주는 형식

```jsx
function User(name, age) {
  //첫 글자는 대문자로
  //this = { };
  this.name = name;
  this.age = age;
  this.sayName = () => {
    console.log(this.name);
  };
}

let user1 = new User("Mike", 30);
let user2 = new User("Lee", 20);
let user3 = new User("Park", 40);
user1.sayName(); // Mike
//new 연산자를 사용해서 호출

//new 함수명 실행 시 알고리즘
//1. 빈객체를 만듦 this = { } this에 할당함
//2. this에 property들을 추가함
//3. 마지막으로 return this; this를 반환함
```
